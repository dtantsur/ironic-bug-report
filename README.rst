Ironic bug reporting
====================

Requires:

* Python 3
* `requests <https://pypi.org/project/requests/>`_

Usage::

    $ ./report.py stats
    Fetching stories from group 75
    Fetching stories from board 67
    Total bugs: 285
     of them untriaged: 420
    Total RFEs: 246
     of them untriaged: 36
